# Linear-Regression
Solve Linear and Non Linear Reggresion  
For PHP.

#### Features
- [x] Solve Linear Regression
- [x] Solve Non Linear Regression
- [x] Plot

### Basic Usage

Download it, then upload to your web server, and navigate to index.html
